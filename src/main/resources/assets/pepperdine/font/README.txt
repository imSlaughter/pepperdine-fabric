This folder should contain Minecraft font provider JSON files.

Example file: sf_medium.json
{
  "providers": [
    {
      "type": "ttf",
      "file": "pepperdine:fonts/sf_medium.ttf",
      "shift": [0.0, 0.0],
      "size": 11.0,
      "oversample": 8.0
    }
  ]
}

The Java font ids are declared in:
font.render.utils.client.pepper.dine.Fonts
